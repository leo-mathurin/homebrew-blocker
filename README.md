# Homebrew Blocker

Homebrew tap for [Blocker.app](https://github.com/leo-mathurin/blocker-releases/releases) — a minimal macOS website blocker with recurring schedules.

## Install

```bash
brew install --cask leo-mathurin/blocker/blocker
```

## Update

```bash
brew upgrade --cask blocker
```

## Uninstall

```bash
brew uninstall --cask --zap blocker
```

## Requirements

macOS 12 (Monterey) or later. Apple Silicon and Intel are both supported.

## Source

The application source lives in a private repository. Signed and notarized release binaries are published at [`leo-mathurin/blocker-releases`](https://github.com/leo-mathurin/blocker-releases).
